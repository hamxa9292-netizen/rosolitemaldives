Connect to your store

This panel edits your live catalog by talking to GitHub. It needs a personal access token — a private key only you have. It's saved only in this browser, never on any server.

How to create your token (one time, ~2 minutes)
  1. On github.com: your profile photo → Settings → Developer settings (bottom of sidebar) → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Name: rosolite-admin. Expiration: 90 days (you'll paste a fresh one when it expires).
  3. Repository access: Only select repositories → choose rosolitemaldives.
  4. Permissions → Repository permissions → Contents → Read and write. Nothing else.
  5. Generate, copy the token (starts with github_pat_), paste it below.
